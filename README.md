# 🛒 MarketUz - Full-Featured Django E-Commerce Platform

MarketUz is a production-ready, fully functional online store built from scratch using Python and the Django framework. The platform provides a seamless user experience, allowing customers to browse catalog items, search for products, manage a dynamic shopping cart, authenticate their accounts, and place orders.

---

## 🚀 Key Features

* **Product Catalog:** Organized by categories with clean slug-based routing and a detailed view for individual products.
* **Session-Based Shopping Cart:** A lightweight, optimized cart logic built on Django Sessions that doesn't put unnecessary load on the database.
* **User Authentication:** Complete user workflow including Sign Up (Registration), Login, and Logout functionality.
* **Order Management System:** Form-driven checkout process that handles guest/authenticated users and structures order details in the database.
* **Smart Search System:** A case-insensitive, real-time product search integrated directly into the global navigation bar.
* **Advanced Admin Dashboard:** A custom-tailored administrative panel allowing store owners to view and manage orders alongside their respective ordered items inline.
* **Modern Responsive UI:** Designed with a sleek, dark-themed Bootstrap 5 navbar and dynamic layout components.

---

## 🛠️ Tech Stack

* **Backend:** Python 3.14+, Django 6.0+
* **Frontend:** HTML5, CSS3, Bootstrap 5
* **Database:** SQLite (Default Django DB)
